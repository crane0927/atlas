# API 合约定义

## 概述

 模块是基础设施模块，不涉及外部 API 接口。本模块主要提供：

1. **Redis 序列化配置**: 内部配置类，不涉及 API 接口
2. **Key 命名规范工具**: 内部工具类，不涉及 API 接口
3. **基础缓存工具**: 内部工具类，不涉及 API 接口

## 内部接口规范

### RedisConfig

**类型**: Spring Configuration Bean
**说明**: 提供 RedisTemplate Bean，供其他模块使用

### RedisKeyBuilder

**类型**: 工具类（Builder 模式）
**说明**: 提供 Key 构建方法，供业务代码使用

### CacheUtil

**类型**: 工具类（静态方法）
**说明**: 提供缓存操作方法，供业务代码使用

## 使用说明

本模块不定义外部 API 接口，所有功能通过工具类和配置类提供给业务模块使用。
